## Quick Setup in VS Code


Given below are instructions on how to set me up.

Prerequisites: VS Code.

1. Open VS Code.
1. Open the project into VS Code as follows:
   1. Click `Open`.
   1. Select the project directory, and click `OK`.
   1. If there are any further prompts, accept the defaults.
1. Alternatively, clone this repository.
1. On the top left corner in VS Code, you should see the `Terminal` option. Click it and select `New Terminal`.
1. After that, follow the instructions on this user guide to add your first task!

## User Guide
### Overview

```
github-activity-fetcher
```

`github-activity-fetcher` is an app designed for you to quickly fetch your latest GitHub activities. It is currently optimized for use via the command-line interface (CLI).

### Fetching GitHub Activity
Fetches a user's GitHub activity.

Format: `github-activity <username>`

Example: `github-activity edwardrl101`

Expected Output:
```angular2html
    - Pushed to edwardrl101/github-activity-fetcher
    - Created branch main in edwardrl101/github-activity-fetcher
    - Pushed to edwardrl101/task-tracker-cli
    - Made edwardrl101/task-tracker-cli public
```
